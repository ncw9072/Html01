** BOX **
=> margin : 요소의 가장자리와 다른 요소와의 간격 (border밖 여백)
=> padding : 요소의 가장자리와 내용 사이의 간격 (컨텐트와 border사이의 여백)
=> border : 요소를 감싸고 있는 외부경계선
=> background-image : 배경이미지
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
** margin, padding **
=> 값은 auto로 하건 여백을 직접 값으로 입력할 수 있습니다.
	( Margin은 음수 사용 가능)
=> 아니면 브라우저의 크기에 따른 비율(%)로 값을 설정할 수 있습니다.
=> margin 1개 : 모두
=> margin 2개 : 위아래, 왼쪽오른쪽
=> margin 4개 : 위, 오른쪽, 아래, 왼쪽(top, right, bottom, left)

** margin-top, margin-right, margin-bottom, margin-left
=> 해당하는 위치의 여백을 설정합니다.
