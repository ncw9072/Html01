** 2. Span **


=> 줄바꿈이 없는 inLine 영역
=> 다른 텍스트와 구별하기 위해 사용
=> 배경색은 글자가 있는곳만 적용
=> 외관 관련 속성은 컨텐츠영역만 적용됨.css(style)
